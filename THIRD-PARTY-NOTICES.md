## chimeranext Labs — Bilingual Format

The Bilingual Format (Human Layer + Agent Layer) used in implementation briefs
produced by the `spec-recommend` and `spike-recommend` skills is a methodology
developed by [chimeranext Labs](https://chimeranext.io). The two-layer structure
ensures briefs are readable by non-engineers (Human Layer) and executable by
AI agents (Agent Layer).

## Plugin Structure

The plugin directory layout (`.claude-plugin/`, `commands/`, `agents/`, `skills/`,
`scripts/`) follows the Claude Code plugin convention and was informed by patterns
