# /ship-snap — Ship to Snap Store (NOT YET IMPLEMENTED)

This command is **not yet implemented**. It lands in **Phase 1** of the roadmap.

For now:
- Use `/app-gtm-release:ship-advisor` to confirm Snap fits your distribution mix.
- For manual Snap publishing: install `snapcraft`, write a `snapcraft.yaml`, and run `snapcraft login && snapcraft register <name> && snapcraft upload <snap>`.
- Reference: https://documentation.ubuntu.com/core/explanation/stores/store-overview/
